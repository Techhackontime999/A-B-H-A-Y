# Contributing to ABHAY

Thank you for your interest in contributing to the ABHAY Protocol! We are building a secure, anonymous platform for truth-seekers, and your help is invaluable.

## üõ° Privacy & Security First
When contributing, always remember that user privacy is our #1 priority.
- **No Tracking**: Do not add analytics, trackers, or cookies that could identify users.
- **No Logging**: Avoid logging IP addresses or user-agent strings.
- **Encryption**: Always verify that client-side data is encrypted before being sent to any server action.

## üöÄ How to Contribute

### 1. Reporting Bugs
- Use the GitHub Issue tracker.
- Describe the bug clearly and provide steps to reproduce.
- Note your environment (Browser, OS).

### 2. Feature Requests
- Open an Issue with the [Feature] tag.
- Explain the "Why" and "How" of the feature.
- Consider how it affects anonymity.

### 3. Pull Requests
1. **Fork** the repository.
2. **Branch**: Create a new branch `feature/your-feature` or `fix/your-fix`.
3. **Code**: Follow the existing style (TypeScript, Tailwind, ShadCN).
4. **Test**: Ensure the app builds and Genkit flows run correctly.
5. **Submit**: Create a PR against the `main` branch.

## üé® Styling & UI
